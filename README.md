This is the user interface for a wireless system that exists in three parts, designed for electricians to remotely test individual circuits in residential and commercial panels prior to receiving certificate of occupancy.
The first part, which is inside of a panel/in contact with the 120/208V will be referred to as ‘PD’ (panel device). The PD is a module that is connected to voltage sources. The part that contains the data, wireless comm, camera, and security suite will be referred to as ‘CH’ (communication hub). The device/app that contains the user interface (UI) has access to all the functions and controls access to the other two parts is called the AP (access point).
